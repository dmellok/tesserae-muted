# Muted — a Tesserae theme pack

Ten mid-tone coloured-canvas themes for Tesserae. One step back from the [Vivid pack](https://github.com/dmellok/tesserae-vivid): meaningfully coloured canvases without crossing into full saturation. Sits between the light-tint [Tonal](https://github.com/dmellok/tesserae-tonal) and [Pigment](https://github.com/dmellok/tesserae-pigment) packs and Vivid's eye-catching maximum-intensity surfaces.

| Theme | Canvas | Tagline |
|---|---|---|
| `muted-brick`    | dusty terracotta | warm dusty terracotta |
| `muted-saffron`  | deep gold | deep dusty gold |
| `muted-olive`    | yellow-green | deep yellow-green |
| `muted-pine`     | forest green | deep forest green |
| `muted-lake`     | teal-blue | muted teal-blue |
| `muted-storm`    | blue-grey | deep blue-grey |
| `muted-mulberry` | magenta-purple | deep magenta-purple |
| `muted-wine`     | red-wine | deep red-wine |
| `muted-terra`    | earth brown | warm earth brown |
| `muted-graphite` | slate grey | deep slate grey |

Each canvas is a HSV ≈ 48% saturation / 78% value — coloured enough to set the dashboard's mood without overwhelming. Text stays dark with a feature tint so it reads on the canvas without crossing into dark-theme styling. The pack's shared supporting accents (gold, bottle green, cobalt, plum, clay) repeat across all ten themes for cohesion.

## Install

Via the catalog: **Tesserae → Settings → Widgets → Browse community widgets**, find "Muted", hit Install.

Requires **Tesserae 0.47.9 or newer** (theme catalog kind).
